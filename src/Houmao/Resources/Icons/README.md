# 图标文件

本目录包含应用程序的图标文件。

## 文件说明

- `houmao.ico` - 应用程序主图标（Windows ICO 格式）

## 如何替换图标

1. 准备一个 256x256 像素的 PNG 图像
2. 使用在线转换工具（如 https://convertico.com/）将 PNG 转换为 ICO 格式
3. 将生成的 ICO 文件重命名为 `houmao.ico`
4. 替换本目录中的文件

## 图标要求

- **格式**: Windows ICO
- **尺寸**: 256x256 像素（推荐），至少包含 32x32 和 16x16 尺寸
- **颜色深度**: 32 位（带透明度）
- **文件大小**: 建议小于 100KB

## 推荐图标设计

- 使用简单的猫咪轮廓
- 主色调：蓝色或紫色
- 背景：透明或纯色
- 风格：现代、简洁、易识别

## 临时解决方案

如果暂时没有图标文件，应用程序仍然可以运行，但会显示默认的 Windows 图标。
